# Vue.js/webpack Tutorial
Vue.jsとwebpackの勉強をするためのリポジトリ

## 状況
- jsのソースコードは src 以下で編集する
- webpack で bundle して public/dist に出力する
- Vue.js の使い方としてはコンポーネントを利用する
- CSS は書いていない
- localStorage を利用
- webpack の設定は最低限の設定をしたので今後改善予定

## 開発の進め方
```
>> yarn run server
```
でファイルの変更を監視してくれる。

プロダクション用の設定などはしていない。
